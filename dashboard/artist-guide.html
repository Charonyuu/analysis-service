插畫家素材指南

合作插畫家素材交付規格與範例參考
主題包（Theme Pack）交付項目
以下為一個完整主題包可能包含的素材類型，所有項目皆為選填，可依合作範圍彈性調整。
角色貼圖（Icon Stickers） 選填
Widget 右下角的裝飾角色
建議尺寸200x200 ~ 300x300 px
格式透明背景 PNG
數量建議 8-16 張一組
命名規則{主題}_{動作}.png
柴犬
貓咪
幽靈
動物們
背景圖（Widget Background） 選填
照片型 Widget 的裝飾邊框背景
建議尺寸450x450 px
格式PNG
數量建議 3-4 張
命名規則{主題}_bg_{n}.png
目前尚無上線的背景圖範例，背景圖為照片型 Widget 外層的裝飾邊框，通常帶有主題風格的花紋或圖案。
DIY 貼圖包（Sticker Pack） 選填
DIY 模式讓使用者自由拼貼的裝飾元素
建議尺寸200x300 px（不等，取決於素材形狀）
格式透明背景 PNG
數量一包通常 8-16 張
命名規則{分類}_{名稱}.png
自然 / 裝飾貼圖範例

ThemeColors 欄位說明
新主題需提供以下色碼欄位：
background
Widget 與頁面的整體背景色
card
卡片 / 內容區塊的背景色
textPrimary
主要文字顏色（標題、重要資訊）
textSecondary
次要文字顏色（副標題、說明文字）
textTertiary
第三層文字顏色（提示、placeholder）
accent
強調色（按鈕、連結、重點裝飾）
buttonForeground
按鈕上的文字顏色
useGlass
是否啟用毛玻璃效果（true / false）
borderColors
邊框漸層色陣列，例如 ["#E0C3FC", "#8EC5FC"]
borderWidth
邊框寬度（數字），例如 1.5
cornerIcon
角落裝飾 SF Symbol 名稱，例如 "sparkle"
cornerIconColors
角落圖示漸層色陣列

交付清單（Delivery Checklist）
完整主題包需要的項目一覽：
  • 角色貼圖 8-16 張（透明 PNG，200-300px）選填
  • 背景圖 3-4 張（PNG，450x450px）選填
  • DIY 貼圖 8-16 張（透明 PNG）選填
  • 主題配色（提供色碼 or 色彩參考圖）建議提供
  • 主題名稱 + 一句話描述
  • 插畫家署名 + 社群連結（IG / Twitter）
所有素材項目皆為選填，可依合作範圍調整
上傳流程說明
素材交付後的處理流程：
  1. 1
    插畫家交付素材（透過雲端硬碟、Email 或其他方式）
  2. 2
    素材上傳到 R2 暫存區，進行格式檢查與尺寸調整
  3. 3
    管理員在 Admin Dashboard 預覽確認素材與配色效果
  4. 4
    確認無誤後推送到正式環境，更新 themes.json 設定檔
  5. 5
    使用者打開 App 自動看到新主題，可直接購買或套用
